## Project 환경 설정
스케폴딩된 프로젝트는 VUE CLI 3를 기준으로 설정되어 있습니다.

### 1. 관련 모듈 설치

```
$ npm install
```

### 2. 로컬에서 development 모드로 개발 시

```
npm run serve
```

### 3. 배포시 production 모드로 빌드 시

```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
